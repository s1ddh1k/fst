# AGENTS.md

## 목표

업비트 KRW 현물 자동매매에서 돈을 번다.
전략을 자동 생성 → 변형 → 백테스트 → 평가 → paper trading까지 사람 개입 없이 연결한다.

## 최적화 목표

파산 위험을 통제한 상태에서 장기 순수익을 최대화한다.
평가 기준: net return, max drawdown, out-of-sample 재현성, 거래 횟수, 비용, 실전 이행 가능성.

## 우선순위

1. auto-research 루프가 실제로 수익나는 전략을 찾게 만든다.
2. 무인 실행, 재시작, 아티팩트 누적.
3. 수익나는 전략을 paper trading까지 연결한다.
4. 단일 전략이 검증된 후, 멀티타임프레임·멀티레짐으로 확장한다.

## 작업 방식

- 묻기보다 바로 구현한다.
- 분석에서 멈추지 말고 자동화, 검증, 재실행까지 이어간다.
- 하나의 전략을 수동으로 다듬기보다, 연구 엔진 자체를 개선한다.
- 결과 보고보다 실제 연구 루프의 품질과 속도를 우선한다.

## 판단 기준

새 작업이 들어오면: "이 변경이 수익나는 전략을 더 빨리 찾게 만드는가?"
아니면 핵심 작업이 아니다.
